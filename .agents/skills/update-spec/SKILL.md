# 명세 업데이트

프로그램 명세가 변경될 때 `proc/spec/`의 관련 문서를 업데이트한다.

## 절차

1. `proc/spec/` 내 기존 문서를 확인한다
2. 변경 사항과 관련된 문서를 식별한다
3. 해당 문서를 업데이트한다. 관련 문서가 없으면 새로 생성한다
4. 변경 내용을 유저에게 요약 보고한다

## 규칙
- 기존 문서가 있으면 수정, 없으면 생성한다
