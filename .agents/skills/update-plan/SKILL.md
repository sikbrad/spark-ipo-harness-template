# 작업 계획 생성/업데이트

`proc/plan/`에 간결한 작업 계획을 생성하거나 기존 계획을 업데이트한다.

## 파일명: `{YYYY-MM-DD}_{내용요약}.md`

## 문서 구조

```markdown
# {작업 제목}

## 목표
한두 줄로 완료 기준

## 작업 항목
- [ ] 항목 1
- [ ] 항목 2
- [ ] 항목 3
```

## 규칙
- 명시적 요청 없이 자동 생성하지 않는다
- 체크리스트를 업데이트하면서 개발한다
- 연관 요청은 같은 문서에 누적한다
