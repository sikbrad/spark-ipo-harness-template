# Threads 바이브코딩 예약 글 작성/발행

## 목표
로컬 바이브코딩 책/꼭지 자료를 바탕으로 개인정보와 회사 내부 상황을 제거한 Threads 체인 글을 작성하고, 말라카 계정에 2026-06-30까지 매일 08:00 예약 발행한다.

## 작업 항목
- [x] 로컬 책/꼭지 자료 위치 확인
- [x] 공개 가능한 주제와 제외 기준 정리
- [x] 2026-06-10부터 2026-06-30까지 21개 Threads 체인 초안 작성
- [x] 개인정보/회사정보/실명/내부 URL 포함 여부 검수
- [x] playwright-cli headful 브라우저로 말라카 계정 로그인/예약발행 가능 여부 확인
- [x] 매일 08:00 예약 설정 및 결과 기록
- [x] 최종 산출물 경로와 예약 결과 보고

## 공개 기준
- 개인 실명, 회사명, 고객명, 내부 프로젝트명, 내부 URL, 계정/조직 상황은 쓰지 않는다.
- 말라카/강의/AX 경험은 특정 조직 사례가 아니라 추상적인 AX 리더 또는 바이브코더 관점으로 바꾼다.
- 문체는 짧은 문장, 구체 장면, 질문형 마무리, 첫 댓글/두 번째 댓글 체인을 따른다.

- Threads 로그인 계정: `@malacca.whale`
