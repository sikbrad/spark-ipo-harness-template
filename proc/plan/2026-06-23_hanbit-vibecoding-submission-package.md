# 한빛 원고투고 제출 패키지 작성 계획

- 요청: 이전 개인 맥락 출판기획서를 더 부연하고, 실제 한빛 원고투고 요구 포맷에 맞춰 제출 가능한 문서로 만든다.
- 추가 요청: 강의이력서/경력기술서 DOCX도 참고해 저자 신뢰 근거를 보강한다.
- 공식 포맷 기준: 한빛 원고투고 웹폼의 필수 항목과 첨부 구조.
- 산출 위치: `data/vibecoding-book/hanbit-submission-2026-06-23/`

## 주의

- 한빛소프트는 게임 퍼블리싱 회사명이므로, 실제 책 투고 브랜드는 `한빛미디어`로 맞춘다.
- 저자 연락처/이메일은 실제 제출 직전에 본인이 입력한다.
- 강의이력서 원본은 개인정보가 포함되어 있으므로 공개 첨부용으로는 요약본만 사용한다.
- 회사 내부정보, 실명, 고객명, 수강생 개인정보는 공개 원고에서 익명화한다.
