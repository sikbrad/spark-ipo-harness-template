## 공개 기준
- 개인 실명, 회사명, 고객명, 내부 프로젝트명, 내부 URL, 계정/조직 상황은 쓰지 않는다.

- 민감 키워드 스캔 결과 직접 식별자 없음. `생김`의 `김`은 일반어 오탐.
- 랜덤 시각표는 모두 07:00~09:00 범위.

## 예약 상태
- 2026-06-09 현재 Threads 웹 예약 UI에서 6월 30일은 선택 후 시간 입력이 유지되지만, 7월 1일은 선택 즉시 예약 달력이 닫히고 시간 입력이 사라진다.
- 따라서 현재 세션에서 7월 1일 이후 예약은 UI 허용 범위 밖으로 판단했다.
- 기존 6월 예약 시간 변경은 프로필/설정/작성 모달/Drafts 진입점에서 예약 목록이나 수정 경로가 노출되지 않아 현재 자동화로 변경 불가.
- 후속 자동화 `schedule-july-threads-posts`를 만들어 매일 오전 09:15에 7월 예약 가능 범위를 다시 확인하고 이어서 예약하도록 설정했다.
