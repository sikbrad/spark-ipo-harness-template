# 포털 고객사 좌표 OSM 연동

## 목표
포털 로컬 DB의 고객사 주소를 OSM Nominatim으로 지오코딩해 고객사에 EPSG:3857 좌표를 저장하고, 고객사 상세 페이지에서 OSM 지도를 표시한다.

## 작업 항목
- [ ] 포털 고객사 스키마, API 계약, 상세 페이지의 현재 주소/상세 렌더링 구조 확인
- [ ] OSM/Nominatim 호출 정책 확인 후 저속/캐시형 백필 방식 설계
- [ ] 고객사 DB/API 타입에 EPSG:3857 lon/lat 필드 추가
- [ ] 고객사 주소 10건만 Nominatim으로 지오코딩해 로컬 DB에 반영
- [ ] 고객사 상세 페이지에 OSM 지도 표시 추가
- [ ] 브라우저에서 10건 중 좌표 저장 및 지도 표출 검증
- [ ] 검증 후 전체 백필/운영 방식 정리
