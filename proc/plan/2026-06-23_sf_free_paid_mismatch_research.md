# SF 무상 조건 vs 포탈 유상 분류 조사

## 목표
Salesforce 스크랩 데이터상 거래조건/결제조건이 무상 계열인데 포탈 DB에서는 `유상(PAID)` 으로 남은 주문을 전수 파악하고, 근거 포함 리서치 문서를 남긴다.

## 작업 항목
- [x] 포탈 DB와 SF 스크랩 원본 위치 확인
- [x] 무상 판정 기준과 포탈 분류 필드 확정
- [x] read-only 대조 쿼리/스크립트로 후보 전수 추출
- [x] 주문별 근거와 패턴을 `proc/research/` 문서로 정리
- [x] 산출물 경로와 집계가 원자료와 일치하는지 재검증
- [x] `FOC` 단독 제외, 한글 `무상`/`무료` 기준 보정 산출물 생성
- [x] 보정 산출물 경로와 집계 재검증
- [x] `무상` 포함 전체 후보를 total 0/PKG/연장/렌탈/AS 등 업무 맥락으로 직접 판정
- [x] 확정 무상 99건과 제외/검토 3,599건 산출물 및 리서치 문서 생성
- [x] Salesforce 스크랩 `grand_total=0` 기준으로 포탈 금액 불일치 1,637건 재산출
- [x] 거래원장/계약잔액용 PKG/프로모션 차감 후보 1,028건 별도 산출
