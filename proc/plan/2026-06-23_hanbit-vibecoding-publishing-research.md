# 한빛 바이브코딩 출판 리서치 계획

- 요청: 한빛 계열 출판사에 `바이브코딩 따라하기` 류의 책을 출판하려면 무엇을 준비해야 하는지 리서치하고 문서화한다.
- 방식: SPARK/IPO에 맞춰 계획은 `proc/plan/`, 조사 결과는 `proc/research/`에 남긴다.
- 범위: 공식 한빛 투고 접점, 한빛소프트/한빛미디어 명칭 구분, 바이브코딩/AI 코딩 도서의 한빛 내 포지셔닝, 제안서 준비 체크리스트.
- 비범위: 실제 투고 제출, 원고 작성, 로컬 자료 인벤토리, 출판기획서 완성본 작성.

## 진행 체크

- [x] 워크스페이스 SPARK/IPO 구조 확인
- [x] `vibecoding-book` 스킬 지침 확인
- [x] 한빛 공식 원고 투고 페이지와 고객센터 안내 확인
- [x] 한빛소프트와 한빛미디어 역할 구분 확인
- [x] 한빛의 바이브코딩/AI 코딩 도서 및 강의 포지셔닝 확인
- [x] 투고 준비 문서 작성

## 검증 기준

- 공식 링크를 우선 근거로 남긴다.
- 사용자 표현인 `한빛소프트`의 오해 가능성을 문서에서 명시한다.
- 다음 행동이 보이도록 제출 경로, 준비물, 권장 포지션, 일정표를 포함한다.
