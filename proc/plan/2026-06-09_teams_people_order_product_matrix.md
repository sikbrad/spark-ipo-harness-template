# Teams 인물 주문/제품/국가 매트릭스 보강

## 목표
정재회, 김규탁, 조소연 아바타 자료에 주문·제품·국가·제품군 기준의 처리 매트릭스를 추가한다. 기존 인물별 판단 규칙을 보존하면서, 특정 주문이나 제품군을 물었을 때 "그 사람이 어떻게 처리했는지" 빠르게 찾을 수 있게 한다.

## 대상
- `data/teams/peoples/김규탁/업무이력/`
- `data/teams/peoples/조소연/업무이력/`
- `data/teams/peoples/정재회/업무이력/`

## 작성 원칙
- 기존 전수장부와 source-index를 근거로 하며 원문 전체를 복사하지 않는다.
- 주문번호, 국가/권역, 제품/제품군, 처리 방식, 보류/확정 조건을 한 행에 둔다.
- 제품군이 불명확하면 억지로 추론하지 않고 `제품군 판단 불가` 또는 업무유형 중심으로 둔다.
- 개인정보, 연락처, 계정, 상세 주소는 쓰지 않는다.

## 검증 메모
- 김규탁, 조소연, 정재회 각각의 `주문-제품-국가-제품군-처리매트릭스.md` 파일이 생성되어 있고 비어 있지 않음을 확인했다.
- 개인 연락처, 계정, 생년월일, 대화방 식별자 같은 민감 식별값이 새 매트릭스와 계획 문서에 없음을 확인했다.
- 문서 내 `output/teams`, `data/company`, `data/teams` 원천 경로가 모두 실제 파일로 역추적됨을 확인했다.
- `git diff --check -- data/teams/peoples proc/plan/2026-06-09_teams_people_order_product_matrix.md` 결과 공백/패치 오류가 없었다.
- `data/`는 `.gitignore` 대상이므로 Git diff에는 데이터 문서 변경이 표시되지 않는다.
