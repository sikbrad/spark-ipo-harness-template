# SIDEX 2026 Outline 게시 계획

## 목표
- `https://outline.doflab.com/doc/sidex2026codex-Pi7l3wBPSd` 문서를 SIDEX 방문 요약 문서로 업데이트한다.
- 같은 문서 아래에 업체별 하위 문서를 만들고, 업체별 사진과 핵심 정보를 정리한다.
- 사용자가 `SIDEX_2026_feedback_memo.xlsx`의 `백인식메모`에 입력한 내용을 사람 인사이트로 간주해 요약 상단에 반영한다.

## 원칙
- 최종 Outline 문서에는 Qwen, VQA, snaps/handouts, 파일 구조, 자동 생성 원리 등 내부 처리 설명을 쓰지 않는다.
- 유저가 볼 문서는 전시회 방문 요약, 업체/분야/제품/주장/현장 인사이트 중심으로 작성한다.
- 이미지가 잘못 회전되어 보이면 원본 또는 게시용 이미지를 보정한 뒤 다시 확인한다.

## 작업 순서
1. Outline API 연결과 대상 문서/컬렉션/문서 ID를 확인한다.
2. 분석 JSON과 사용자가 채운 엑셀 메모를 병합해 게시용 데이터셋을 만든다.
3. 이미지 방향을 점검하고 게시용 이미지 세트를 만든다.
4. Outline 상위 요약 문서를 업데이트하고 업체별 하위 문서를 생성/갱신한다.
5. 브라우저로 대상 문서와 하위 문서/이미지 렌더링을 확인한다.
