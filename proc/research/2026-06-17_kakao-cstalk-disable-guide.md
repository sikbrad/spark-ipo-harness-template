# 카카오 상담톡 비활성화 절차 문서

작성일: 2026-06-17  
범위: 문서 작성만 수행. 실제 카카오, MatrixCloud, Zendesk 설정 변경은 하지 않음.

## 1. 결론

상담톡은 카카오 채널 관리자센터에서 단순히 ON/OFF 하는 일반 1:1 채팅 기능이 아닙니다. 카카오 공식 가이드 기준 상담톡은 상담 메시지 API와 카카오 시스템 접근 권한 상품이며, 실제 상담 화면은 공식 딜러사/중계사 또는 외부 상담 솔루션에서 제공합니다.

따라서 상담톡을 비활성화하려면 보통 아래 순서가 맞습니다.

1. 현재 상담톡 중계사/딜러사를 확인한다.
2. 중계사/딜러사에 상담톡 연동 해제 또는 계약 종료를 요청한다.
3. 홈페이지, 앱, 챗봇, 채널 메뉴에 노출된 `카카오톡 상담하기` 버튼/링크를 제거하거나 다른 상담 경로로 바꾼다.
4. 상담톡 해제가 확인된 뒤, 필요하면 카카오 기본 `1:1 채팅`을 켠다.
5. Zendesk/Matrix Chat 앱, 트리거, 태그, 매크로는 신규 인입이 멈춘 것을 확인한 뒤 정리한다.

현재 DOF 화면에는 Zendesk 우측 패널에 `Matrix Chat1.1`이 보이고 티켓 태그에 `matrixchat`, `kakaotalk`이 붙어 있으므로, 실무상 1차 연락처는 **MatrixCloud**일 가능성이 높습니다.

MatrixCloud 공개 FAQ 기준으로는 카카오 상담톡 연동 해제는 직접 처리할 수 없고, 중계사에 연동 해지 신청이 필요합니다. MatrixCloud의 경우 고객지원 메일 `support@matrixcloud.kr`로 **회사명과 채널명**을 기재하여 해제 요청하라고 안내되어 있습니다.

## 2. 먼저 구분해야 하는 것

상담톡 비활성화에는 최소 4개의 레이어가 있습니다. 이 중 하나만 끄면 운영이 꼬일 수 있습니다.

| 레이어 | 의미 | 누가 처리하나 | 비고 |
| --- | --- | --- | --- |
| 카카오 상담톡 계약/API 연동 | 카카오 채널과 상담톡 중계사의 연결 | 상담톡 공식 딜러사/중계사 | 직접 ON/OFF가 아니라 해제 요청 필요 |
| Matrix Chat/Zendesk 연동 | 카카오 상담을 Zendesk 티켓으로 만드는 연동 | MatrixCloud + Zendesk 관리자 | 상담톡 해제 전 임의 삭제 금지 |
| 고객 진입 버튼 | 홈페이지/앱/챗봇의 `카카오톡 상담하기` 버튼 | 웹/앱/챗봇 운영 담당자 | 상담톡 중단 시 제거 또는 대체 필요 |
| 대체 상담 채널 | 기본 1:1 채팅, 전화, 이메일, 웹폼 등 | 채널/CS 운영자 | 고객에게 새 문의 경로 공지 필요 |

## 3. 공식 근거 요약

### 카카오 상담톡 공식 가이드

- 상담톡은 `카카오톡 상담하기` 버튼, 채널 검색, 메시지 수신 화면 등에서 상담 요청 시 시작되는 상담 메시지 API 상품입니다.
- 상담톡은 고객센터용 채팅 상담 시스템 자체를 제공하는 상품이 아니라, API와 카카오 시스템 접근 권한을 제공하는 구조입니다.
- 상담톡을 정식 사용하는 시점부터 기존 카카오톡 채널 관리자 웹/앱의 채팅 메뉴는 비활성화되고, 기존 채팅 이력과 이용자별 메모는 더 이상 조회할 수 없게 된다고 안내되어 있습니다.
- 상담톡 사용을 중단하면 `카카오톡 상담하기` 버튼은 정상 작동하지 않으므로, 홈페이지 등에 노출된 버튼을 제거해야 한다고 안내되어 있습니다.

### MatrixCloud FAQ

- MatrixCloud는 카카오 상담톡 연동 해제를 직접 처리할 수 없고, 카카오 상담톡 중계사에 연동 해지 신청을 해야 한다고 안내합니다.
- MatrixCloud 이용 고객은 `support@matrixcloud.kr`로 회사명과 채널명을 기재하여 해제 요청하라고 안내되어 있습니다.

## 4. 비활성화 전 확인 체크리스트

### 4-1. 채널/계약 정보

- 카카오 채널명
- 채널 홈 ID 또는 검색용 ID
- 사업자명
- 카카오 비즈니스 채널 인증 여부
- 상담톡 공식 딜러사/중계사 이름
- MatrixCloud 계약 여부
- Zendesk 서브도메인
- 현재 상담톡 운영 담당자/계약 담당자
- 해제 희망일과 시간

### 4-2. 현재 고객 진입 경로

아래 위치에 `카카오톡 상담하기` 또는 상담톡 링크/버튼이 있는지 확인합니다.

- 회사 홈페이지 메인/고객지원/제품 페이지
- 쇼핑몰 또는 주문 페이지
- 모바일 앱
- 카카오 채널 홈의 버튼/메뉴
- 카카오 챗봇 시나리오
- 카카오 채팅방 메뉴
- 광고 랜딩페이지
- QR 코드, 오프라인 안내문, 명함, 이메일 서명

상담톡 해제 후 이 버튼이 그대로 남아 있으면 고객이 눌러도 상담 연결이 실패하거나 잘못된 경로로 들어갈 수 있습니다.

### 4-3. Zendesk/Matrix Chat 운영 정보

- Zendesk에서 카카오톡 상담 티켓을 구분하는 태그: 예 `kakaotalk`, `matrixchat`
- Matrix Chat 앱 이름: 예 `Matrix Chat1.1`
- 카카오톡 상담 전용 그룹/담당자
- 카카오톡 상담 전용 트리거/자동화
- 매크로, SLA, 보고서, 대시보드
- 진행 중 티켓 수
- 최근 30일 상담량
- 상담 이력 보관 정책

## 5. 권장 절차

### Step 1. 내부 결정

먼저 “상담톡을 완전히 끌 것인지” 또는 “Zendesk/Matrix Chat 운영만 잠시 중단할 것인지”를 구분해야 합니다.

권장 결정 항목:

- 상담톡 계약 자체 해제 여부
- 해제 후 기본 1:1 채팅으로 전환할지 여부
- 고객에게 공지할 대체 문의 경로
- 기존 Zendesk 상담 이력 보관 방식
- 해제 희망일
- 롤백 가능 기간

### Step 2. MatrixCloud/중계사에 해제 요청

DOF 현재 화면 기준으로는 MatrixCloud에 먼저 문의하는 것이 합리적입니다. 단, 실제 계약 중계사가 MatrixCloud가 아니면 해당 중계사 또는 공식 딜러사에 요청해야 합니다.

요청 시 포함할 정보:

- 회사명
- 사업자명
- 카카오 채널명
- 채널 홈 ID 또는 검색용 ID
- Zendesk 서브도메인
- 상담톡 연동 해제 희망일
- 해제 목적: 예 기본 1:1 채팅 전환, 상담 채널 정리, 비용 절감 등
- 해제 후 기존 상담 이력 조회 가능 여부 문의
- 해제 후 카카오 기본 1:1 채팅 메뉴 복구 예상 시점 문의
- 해제 취소/롤백 가능 여부 문의

### Step 3. 고객 진입 버튼 제거 또는 대체

카카오 공식 가이드는 상담톡 사용 중단 시 `카카오톡 상담하기` 버튼 제거가 필요하다고 안내합니다.

해제일 전에 아래 중 하나로 정리해야 합니다.

- 상담톡 버튼 제거
- 버튼을 기본 카카오 채널 1:1 채팅 링크로 교체
- 버튼을 Zendesk 웹폼/전화/이메일 문의로 교체
- 챗봇의 상담직원 연결 경로를 기본 1:1 채팅 또는 다른 경로로 수정
- 고객 안내 문구 변경

### Step 4. 진행 중 상담 정리

해제 전에 Zendesk/Matrix Chat에 남아 있는 진행 중 상담을 정리합니다.

- 진행 중 카카오톡 티켓 목록 추출
- 긴급/미해결 티켓 담당자 배정
- 고객에게 후속 연락 경로 안내
- 상담 종료 처리
- 필요한 경우 상담 이력 백업

### Step 5. 해제 확인

중계사에서 상담톡 해제 완료를 통보한 뒤 아래를 확인합니다.

- 신규 카카오톡 상담이 Zendesk 티켓으로 생성되지 않는가
- 고객이 기존 상담톡 버튼을 눌렀을 때 실패/혼선이 없는가
- 카카오비즈니스 파트너센터 또는 채널 관리자센터에서 기본 `1:1 채팅` 메뉴가 보이는가
- 기본 1:1 채팅을 쓸 계획이면 `채팅설정`에서 사용 여부, 운영 시간, 안내 문구를 설정할 수 있는가
- 기존 Zendesk 티켓/댓글/첨부파일 조회가 유지되는가

### Step 6. Zendesk 정리

상담톡 해제 완료와 신규 인입 중단을 확인한 뒤에만 Zendesk 정리를 검토합니다.

정리 대상:

- Matrix Chat 앱 비활성화 또는 권한 제한
- `kakaotalk`, `matrixchat` 태그 기반 트리거 비활성화
- 상담톡 전용 자동화, SLA, 뷰, 보고서 보관 처리
- 신규 티켓 생성 조건 변경
- 담당자 그룹/매크로 정리

주의:

- Zendesk 앱을 바로 삭제하지 말고 먼저 비활성화 또는 접근 제한으로 테스트하는 편이 안전합니다.
- 기존 상담 이력 보존이 필요하면 앱 삭제, 필드 삭제, 태그 삭제 전에 보존 정책을 확정해야 합니다.

## 6. 기본 1:1 채팅으로 전환할 경우

상담톡 해제 후 기본 1:1 채팅을 쓰려면 아래 설정을 확인합니다.

1. [카카오비즈니스 파트너센터 로그인](https://business.kakao.com/login)
2. 운영 채널 선택
3. `채팅` 또는 `1:1 채팅` 메뉴 확인
4. 관리자 추가인증 진행
5. 채팅 사용 ON
6. 채팅 가능 요일/시간 설정
7. 채팅 불가능 안내 메시지 설정
8. 채팅 안내 메시지 설정
9. PC/모바일 알림 설정
10. 자주 쓰는 답변, 라벨, 담당자 운영 기준 설정

단, 상담톡 해제가 완료되지 않은 상태에서는 기본 1:1 채팅 메뉴가 비활성화되어 있을 수 있습니다.

## 7. 해제 요청 메일 초안

아래 문안을 MatrixCloud 또는 실제 상담톡 중계사에 보낼 수 있습니다.

```text
제목: 카카오 상담톡 연동 해제 요청 - [회사명] / [카카오 채널명]

안녕하세요.

[회사명]의 카카오 상담톡 연동 해제를 요청드립니다.

1. 회사명:
2. 사업자명:
3. 카카오 채널명:
4. 카카오 채널 홈 ID 또는 검색용 ID:
5. Zendesk 서브도메인:
6. 현재 연동 솔루션: Matrix Chat / Zendesk
7. 해제 희망일:
8. 해제 사유:
9. 해제 후 운영 예정 방식: 기본 1:1 채팅 / 다른 상담 채널 / 미정

확인 부탁드릴 사항은 아래와 같습니다.

- 상담톡 연동 해제 가능 일정
- 해제 후 카카오 채널 관리자센터/카카오비즈니스 파트너센터의 기본 1:1 채팅 메뉴 복구 여부와 예상 시점
- 기존 Zendesk/Matrix Chat 상담 이력 조회 가능 여부
- 해제 후 롤백 가능 여부
- 해제 전 고객사에서 제거해야 하는 버튼, 링크, 챗봇 설정 항목
- 추가 비용, 위약금, 정산 이슈 여부

감사합니다.
```

## 8. 고객 공지 문구 예시

### 기본 1:1 채팅으로 전환하는 경우

```text
카카오톡 상담 채널이 변경됩니다.

기존 상담톡 상담은 [전환일]부터 카카오톡 채널 기본 1:1 채팅으로 운영됩니다.
문의는 동일한 카카오톡 채널에서 메시지를 보내주시면 됩니다.
상담 가능 시간은 [운영 시간]입니다.
```

### 전화/이메일/웹폼으로 전환하는 경우

```text

이후 문의는 아래 경로를 이용해 주세요.
- 전화:
- 이메일:
- 문의 접수 페이지:

기존 상담 건은 순차적으로 처리하겠습니다.
```
